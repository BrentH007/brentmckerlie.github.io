ReCall OCR PDF Highlighter

Desktop utility to search PDFs (including scanned documents) and automatically highlight matches

View on GitHub Features & Documentation

Main Interface

Clean, intuitive interface with PDF viewer, search panel, and highlighting controls. Supports both text-based and OCR search.

Search & Highlight

Real-time search with automatic highlighting. Supports multiple search terms with different colors for easy identification.

OCR Processing

Batch OCR processing for PDF documents. Progress tracking and quality settings for optimal text extraction.

Key Features & Documentation

Core Functionality

  • PDF Text Search: Instant search through native PDF text content
  • OCR Integration: Powered by Tesseract for scanned document processing
  • Auto Highlighting: Automatic highlighting of search matches with customizable colors
  • Batch Processing: Process multiple documents simultaneously

Technical Stack

  • Python 3.8+: Core application logic
  • PyQt6: Modern desktop GUI framework
  • pytesseract: OCR text extraction engine
  • pdfplumber: PDF text and metadata extraction
  • PyMuPDF: PDF rendering and annotation

Installation & Usage

  • Requirements: Windows 10+, macOS 10.14+, or Linux
  • Installation: Standalone executable or Python package
  • Supported Formats: PDF, images (JPG, PNG, TIFF)
  • Export Options: Annotated PDF, plain text, CSV reports
