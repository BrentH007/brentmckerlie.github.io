Key Features & Documentation

Core Functionality

  • Multi-Format Import: Support for CSV, Excel, and QuickBooks document formats
  • Smart Categorization: Automatic transaction categorization with manual override options
  • Expense Analysis: Comprehensive breakdown of income and expenses with visual charts
  • Cost Reduction: AI-powered suggestions for reducing expenses and optimizing spending
  • Report Generation: Detailed financial reports with export capabilities

Technical Stack

  • Godot 4: Modern game engine for cross-platform desktop application
  • GDScript: Primary scripting language for application logic
  • CSV Parser: Custom parsing engine for bank statement formats
  • SQLite: Local database for transaction storage and management
  • Chart.js Integration: Interactive charts and data visualization

Supported Formats & Features

  • Import Formats: CSV, XLSX, XLS, QBO, QFX files
  • Bank Support: Major banks and credit unions with standard formats
  • Export Options: PDF reports, CSV data export, summary sheets
  • Security: Local data storage with encrypted transaction details
  • Backup: Automatic backup and restore functionality

Installation & Usage

  • Platforms: Windows 10+, macOS 10.14+, Linux (Ubuntu 20.04+)
  • Installation: Standalone executable with installer package
  • Data Import: Wizard-guided import process with format detection
  • User Guide: Built-in help system with video tutorials
